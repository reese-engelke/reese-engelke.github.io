Project 01: Image Processing Accelerator

Built a custom Sobel Edge Detection kernel in Verilog for a Cyclone IV FPGA. Optimized for 60fps at 1080p.

// Sample RTL snippet
always @(posedge clk) begin
    if (reset) 
        pixel_out <= 8'h00;
    else
        pixel_out <= (gx + gy) >> 1;
end
